Project Overview

Built a technical SEO landing page in WordPress using a child theme and custom block template. The page was structured to support clear service positioning, conversion-focused CTAs, and a simple inquiry workflow through WPForms. I also configured page-level SEO metadata in Rank Math and designed the layout to be clean, responsive, and suitable for small and service-based businesses.
